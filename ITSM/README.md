# ServiceNow ITSM — IT Service Management

ITSM (IT Service Management) is ServiceNow's flagship module and the foundation on which most other platform capabilities are built. It applies ITIL best practices to manage the full lifecycle of IT service requests, incidents, problems, and changes — replacing ad-hoc email-based IT support with structured, trackable, SLA-governed workflows.
