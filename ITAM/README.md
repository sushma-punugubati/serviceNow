## Quick Reference

```
ITAM = Hardware + Software + Cloud + Mobile asset management
HAM  = Physical assets (alm_hardware table)
SAM  = Software licenses (alm_license table)
Asset lifecycle = Planning → Procurement → Receiving → Deployment → Maintenance → Retirement
CMDB link = alm_asset ← linked to → cmdb_ci
Publisher Packs = Pre-built license models for major vendors
License Workbench = Central compliance dashboard
Discovery = Populates asset data automatically
```
