# ServiceNow AI — Artificial Intelligence & Now Assist

ServiceNow's AI portfolio spans generative AI (Now Assist), conversational AI (Virtual Agent), machine learning (Predictive Intelligence), autonomous agents (AI Agents), governance (AI Control Tower), and AI-native development tooling. Together, these capabilities transform how employees interact with IT, HR, and business services — and how developers build on the platform.

| Capability | What it does |
|-----------|-------------|
| **Now Assist** | Generative AI for agents, employees, and developers (LLM-powered) |
| **Virtual Agent** | Conversational chatbot with NLU for self-service deflection |
| **Predictive Intelligence** | ML-based auto-classification and routing |
| **AI Agents** | Autonomous multi-step agents that execute workflows without human intervention |
| **AI Control Tower** | Governance, monitoring, and safety controls for all AI features |

| **Document Intelligence** | AI-powered document extraction and classification |
| **Health Log Analytics** | AIOps — ML anomaly detection in logs and metrics |

| **Now Assist Search** | Semantic search across knowledge and catalog |
