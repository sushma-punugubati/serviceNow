# ServiceNow SAM Pro — Software Asset Management

ServiceNow SAM Pro (Software Asset Management) tracks the full lifecycle of software licenses — from purchase through deployment, usage monitoring, and reclamation. It answers the core compliance question: "How many licenses do we own, how many are installed, and are we over- or under-licensed?"

SAM Pro connects Discovery (what is installed) with Contract/Procurement data (what is purchased) to give a continuous, automated compliance position — replacing the reactive, audit-driven approach most organizations use.
