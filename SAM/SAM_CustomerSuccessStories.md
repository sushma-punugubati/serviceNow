## Case Study 1: Global Financial Services Firm — Avoiding a $12M Microsoft Audit

**Organization:** Fortune 500 financial services company, 45,000 employees
**Challenge:** Microsoft EA renewal was approaching. The organization had never done a formal software reconciliation and had no idea what their compliance position was. Microsoft had notified them of an upcoming True-Up audit.

**Solution Implemented:**
- Deployed SAM Pro with the Microsoft Publisher Pack
- Integrated SCCM data (38,000 endpoints) with SAM Pro normalization
- Ran initial reconciliation 90 days before the True-Up date
- Identified 3,200 over-installed Office seats and 800 under-installed SQL Server licenses
- Reclaimed unused Office licenses from machines with no login in 180 days
- Purchased additional SQL Server licenses proactively before the audit

**Results:**
- Avoided $12M in potential back-payment liability identified during pre-audit reconciliation
- Reclaimed 1,800 Office licenses (saving $540K in renewal cost)
- True-Up audit completed in 3 hours vs. the typical 3-month discovery process
- Ongoing SAM Pro management reduced annual True-Up preparation time from 6 weeks to 2 days

## Case Study 2: Healthcare Network — 22% Software Cost Reduction Through Reclamation

**Organization:** Regional healthcare network, 12 hospitals, 28,000 employees
**Challenge:** Software costs had grown 40% over 3 years with no corresponding growth in headcount. Leadership suspected significant over-purchasing but had no visibility into actual usage.

**Solution Implemented:**
- Deployed SAM Pro with SaaS License Management for cloud applications
- Connected Zoom, Microsoft 365, Adobe Creative Cloud, and Epic to SAM Pro
- Set 60-day no-usage threshold for reclamation (90 days for clinical applications)
- Implemented automated reclamation notifications through Employee Center

**Results:**
- Identified 4,200 Zoom seats with no login in 60+ days (out of 18,000 total seats)
- Reclaimed 3,800 Zoom seats after user notification and response period
- Identified 2,100 unused Adobe Creative Cloud seats (had been purchased for a one-time project and never reclaimed)
- Total first-year software cost reduction: **22% ($3.2M annual savings)**
- SaaS spend visibility dashboard created for CFO: real-time view of seat utilization by application

## Case Study 3: Manufacturing Company — Oracle Audit Readiness

**Organization:** Global manufacturing company, 60,000 employees across 18 countries
**Challenge:** Received an Oracle License Management Services (LMS) audit notification. Oracle audits are notoriously complex — particularly for database products running on virtual environments where license counting rules are very specific.

**Solution Implemented:**
- Deployed SAM Pro with Oracle Publisher Pack
- Configured Oracle Database license counting rules for virtualized environments (VMware processor factor tables, Oracle hard partitioning vs. soft partitioning rules)
- Connected VMware vCenter data to CMDB for accurate virtual host configuration data
- Ran pre-audit reconciliation specifically for Oracle Database and Oracle Middleware

**Results:**
- Pre-audit reconciliation identified under-licensing of 24 Oracle Database processor licenses due to incorrect VMware partition configuration
- Proactively purchased additional licenses before Oracle LMS team arrived — avoided audit penalties
- Identified 180 over-licensed Oracle Application Server licenses — negotiated a credit on the renewal
- Net audit outcome: **neutral** (vs. typical Oracle audit settlements of $1M-$5M for similar-sized organizations)
- Permanent Oracle compliance monitoring established — never surprised by audit again

## Case Study 4: Technology Company — SaaS Sprawl Reduction

**Organization:** Mid-size technology company, 8,000 employees, 100% cloud-based IT
**Challenge:** SaaS application count had grown to 340+ tools across the organization. IT had no visibility into which tools were actually being used vs. which were "shadow IT" subscriptions that were expiring or accumulating unused seats.

**Solution Implemented:**
- Deployed SAM Pro SaaS License Management
- Connected the 40 highest-spend SaaS applications via IntegrationHub
- Built a SaaS portfolio dashboard showing cost, seat count, and utilization per application
- Implemented a SaaS governance process: new SaaS purchases over $10K required a SAM review before approval

**Results:**
- Identified 14 SaaS applications with less than 20% active seat utilization
- Cancelled or renegotiated 8 applications (saving $890K annually)
- Reduced total SaaS application count from 340 to 210 through consolidation
- SaaS governance process prevented $2.1M in duplicate/redundant purchases in the first year
- CISO gained visibility into all active SaaS applications — resolved a long-standing security concern about unknown data processors

## Case Study 5: Government Agency — FITARA Compliance and Asset Visibility

**Organization:** US Federal Government agency, 15,000 employees
**Challenge:** Required to achieve FITARA (Federal IT Acquisition Reform Act) compliance which mandates software asset inventory and license compliance tracking for all IT spend.

**Solution Implemented:**
- Deployed SAM Pro integrated with the agency's existing ServiceNow ITSM platform
- Configured import sets for legacy software inventory data from 12 different system sources
- Built Publisher Packs normalization for all COTS (Commercial Off-The-Shelf) software in the portfolio
- Created FITARA compliance dashboard showing inventory completeness and compliance position by system category

**Results:**
- Achieved FITARA compliance attestation within 6 months of deployment
- Software inventory completeness improved from **31% to 94%** of known software titles
- Identified $4.7M in duplicate software licenses across agencies sharing the platform
- Annual IG audit preparation reduced from 8 weeks to 3 days with automated report generation
- SAM Pro became the authoritative source for software spend reporting to Congress
