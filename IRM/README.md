## Quick Reference

```
IRM = Governance + Risk + Compliance + Audit + Vendor Risk + Resilience
Key concept    = Inherent Risk vs Residual Risk (after controls)
Test Once, Comply Many = One control → satisfies multiple regulations
Risk Appetite  = Overall level of risk org will accept
Risk Tolerance = Specific thresholds that trigger action
KRI            = Key Risk Indicator (early warning metric)
Control types  = Preventive | Detective | Corrective
Entity         = Org unit (business unit, location, vendor)
CSDM alignment = Links risks/controls to business services
```
